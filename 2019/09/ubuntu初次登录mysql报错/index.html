Ubuntu初次登录MySQL报错

2019.9.1 发布在 问题及解决方法, MySQL

初次登录显示1698(28000)错误代码

1698(28000)错误代码是因为没有使用管理员身份，我们使用sudo su切换到管理员权限即可解决问题

sudo su   #切换管理员权限
# 初次登录没有密码
mysql -u root -p
