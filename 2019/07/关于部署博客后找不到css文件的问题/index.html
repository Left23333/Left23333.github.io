关于部署博客后找不到css文件的问题

2019.7.31 发布在 博客搭建及问题

-buildDrafts命令在Windows下不适用导致的部署无法找到css文件

类Unix开发环境搭建比Windows方便太多，因此网络上多是基于Mac及Linux的个人博客搭建及部署教程。类Unix操作系统与Windows系统就有很大差别，在Windows环境下难免遇到各种问题及困难。

在mac中用-buildDrafts命令来创建草稿，在Windows中不适用，会导致无法产生样式，所以我们使用去掉-buildDrafts的命令

hugo server -t 主题文件夹名 –buildDrafts
hugo server -t 主题文件夹名

在创建public文件夹时如果末尾有-buildDrrafts，部署带GitHub仓库便会找不到css文件。因此我们在这里也把-buildDrafts去掉

hugo  -–theme=主题文件名 -–baseUrl=“https://youer_name.github.io” –buildDrafts
hugo  -–theme=主题文件名 -–baseUrl=“https://youer_name.github.io”
